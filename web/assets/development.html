Development

Roadmap and full release history. Issues and pull requests welcome on GitHub.

Roadmap

  • next
    Persistent multi-file pairing Today the live session ends after the first file. Keeping it open lets the same sender drop a second file without re-pairing.
  • next
    Receiver-side accept prompt One-tap confirmation with the filename and size before saving.
  • soon
    Multi-arch Docker image The relay image ships linux/amd64 today. Adding native arm64 helps self-hosters on Apple Silicon and Raspberry Pis.
  • soon
    TURN relay for tricky networks A few firewalls block peer-to-peer outright. An optional TURN relay closes the gap.
  • later
    LAN device auto-discovery Devices on the same network or tailnet pick each other from a list, instead of typing a passphrase.
  • later
    Configurable mailbox limits The mailbox is hard-coded to 500 MB and 10 minutes today. Self-hosters should be able to set their own.
  • later
    Storage caps A total-bytes ceiling per relay with backpressure on new uploads, so self-hosted relays can't be abused.

Got an idea? Open an issue.
