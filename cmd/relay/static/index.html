AIRPIPE

Send a file to anyone, peer-to-peer through your browser. Encrypted. Self-hostable.

Send a file → or upload to a 10-min mailbox

The relay only ever sees encrypted signaling, never your file.

Run your own relay

One container. Bundles the landing, the browser pages, and the install script. Or use this one. Either way, the relay never sees your files.

Open source, MIT. github.com/Sanyam-G/Airpipe

docker run -p 8080:8080 ghcr.io/sanyam-g/airpipe-relay cp
